# geoexpert
Learn geography with interactives and quizzes.

Welcome to Geoexpert!
This website lets you interact with maps and quiz yourself on your geography knowledge.

The home, or index, page lets you view the variety of maps and hover over them to see what they are. At the top of the website, there is a navigation bar that allows you to choose between the different quiz options you would like to take. When you click a quiz, it will ask whether you would like to start it. This will start a timer, the task, and the score counter. The user is expected to click the part of the map that corresponds to the requested country. When finished, the quiz will end, and the final score can be found at the top.
