# geoexpert
Learn geography with interactives and quizzes.

Welcome to Geoexpert!
This website lets you interact with maps and test your geography knowledge.

HTML, CSS, and JavaScript were used for this project on w3schools.

The home, or index, page lets you view the variety of maps and hover over them to see what they are. At the top of the website, there is a navigation bar that allows you to choose between the different quiz options you would like to take. When you click a quiz, it will prompt you to start it. This will start a timer, the task, and the score counter. The user is expected to click the map area corresponding to the requested country. When finished, the quiz will end, and the final score can be found at the top.
